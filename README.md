# Young's Modulus and Poisson's Ratio of Germanium Using DFT

## Overview

This project uses Density Functional Theory (DFT) calculations performed with Quantum ESPRESSO to determine the Young's modulus and Poisson's ratio of germanium.

The workflow includes:
- Variable-cell relaxation (vc-relax)
- Strain-based simulations under tension and compression
- Linear regression analysis of stress-strain data
- Determination of Young's modulus and Poisson's ratio
- Scientific visualization using Python

Repository documentation is currently being expanded.
